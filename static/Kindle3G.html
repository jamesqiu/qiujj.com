Kindle3入手折腾日记

说是日记，其实也就零星时间折腾了2天，记下来给还在折腾以及将要折腾的人做个参考。我从淘宝上销量第一的卖家购的全新Kindle3 3G+Wifi广告版套餐3（国产卡扣皮套，不是绑4个角的那种），北京凌晨下单当天下班前就送到。买Kindle是想给自己个多读几本书的理由，上班及在家都对着计算机液晶屏，手持不便，眼也受不了；4寸手机屏太小其实手持也很不舒服，除了中午吃饭等上菜间隙surfing一下也懒得看，最大的用途也就是平时做相机出差时做wifi热点。kindle这2天用的基本感受：

l  eink电子墨水很有特点，用来看书那是绝了，但上网游戏看图因为刷屏慢没彩色，不行；

l  耗电少待机时间长这个是杀手级特性，不用关机不用老惦记着带充电器数据线这些杂碎；

l  外放mp3音效足够好，超过一般笔记本和手机，耳机这个杂碎也可不用惦记了；

l  5向键很不习惯，快速移动不爽；输入数字符号也是个麻烦事；

废话少说，切入主题，没装“多看”中文系统，因为据说不支持上网，那我这3g不是白买了吗，用的少但要的时候你得有，Kindle推送的广告也没必要干掉，自已要找几张这么顺眼的屏保还真不容易。要比较舒服地看中文、传文件，如下几个破解是必须的：kindle-jailbreak、kindle-fonts、kindle-usbnetwork，主要参考 http://irising.me/tag/kindle/ 等博文，下面且听我一一道来：

（一） 开机上套注册

知道开Wifi或3G，会用[Sym]键输入符号数字就可以从Kindle开始注册了，kindle管3G和wifi都叫“wireless”，自动切，先wifi，没有了找3g，再没有找gprs，名字和邮件前缀你根据爱好输入，不用担心和别人重复，需要一个US地址，从你名片夹中顺便找张老美的名片来借用一下即可，还需要填一个支持EMV的信用卡，国内信用卡应该都有Visa标没问题，完了你从PC上登录https://www.amazon.com/gp/digital/fiona/manage 就能看到你的Kindle了，除了能直接看到的 xxx@kindle.com邮箱，还会自动有一个xxx@free.kindle.com邮箱，在这个界面上记着把 [Personal Document Settings] / [Maximum Personal Document Charge] / [Maximum Charge Limit] 设成0，免得担心不注意被收$了，在同一个页面的[Approved Personal Document E-mail List]下加上几个你常用的邮箱地址以便之后给自己email传书。顺便说一下我理解的Kindle的3G收费策略：

l  美国内所有免费

l  美国外订阅kindle官方的newspaer、magazine、blog费用每周$4.99, 每月约 140 RMB；

l  美国外email传送个人文档（非购买书籍）到kindle, 每兆 $0.99, 100 M 就约 700 RMB

l  美国外email传送个人文档到@free.kindle.com邮箱免费

l  美国外上网浏览免费

（二） 破解中文字体

先到http://www.mobileread.com/forums/showthread.php?t=88004 把如下几个文件的当下最新版下回来：

l  kindle-jailbreak-0.9.N.zip (95.1 KB, 624 views)

l  kindle-fonts-4.8.N-k3.zip (19.48 MB, 1457 views)

l  kindle-usbnetwork-0.34.N.zip (4.04 MB, 944 views)

最后一个之后用于破解wifi传文件。解压后连上kindle数据线，先把适用于3G广告版的update_jailbreak_0.9.N_k3g-3.2.1_install.bin这一个文件拷到kindle根目录下后在kindle上按[Home] / [Menu] / [Settings] / [Menu] / [Update your kindle]即可，重启较慢，耐心等待。

之后就可以更新字体文件了，解压kindle-fonts-4.8.N-k3.zip，把update_fonts_4.8.N_k3g_install.bin文件copy到kindle根目录下，在kindle上按[Home] / [Menu] / [Settings] / [Menu] / [Update your kindle]即可，重启较慢，耐心等待。

因为上面的字体把原系统的英文字体也替换了，我觉得没有原系统的英文字体好看，再下载 http://dl.dbank.com/c0mu0y3qt8 上的“原系统英文字体.rar”，解压后覆盖替换kindle上/linkfonts/fonts下的同名文件，因为不是update_xxx.bin形式的文件，你得自己想办法让kindle重启，办法就是复制文件autoreboot，改名为reboot后，拔数据线就自动重启了。

现在看中文文件名、中文内容就都不会出现“口口”等非正常情况了，英文还是原来那种美观的字体，有网友还折腾换其他中文字体，这看个人爱好，反正我这种还在用Fixedsys古老字体的人是觉得够美观了。

（三） 安装中文字典

中文字典我把淘宝CD上带的xiang-encn.mobi（Xiang's dictionary V4）字典文件复制到K3的documents目录下并在[Home] / [Menu] / [Settings] / [Menu] / [Change primary dictionary]勾选上即可，这个字典支持复数、首字母大写等变形的取词，比较方便（21shiji、ncce-ec不支持）。

（四） 看图片听音乐

图片copy到/pictures/ 下自建的目录里面Alt + z就可以显示到Home页面上的列表里，我传了几幅书法字帖和几张照片上去，显示效果还不错。mp3拷贝到/music下即可，从[Home] / [Menu] / [Experimental] / [Play mp3]播放，操作说明也写在这里，之前提到，kindle外放音效算相当出色。

（五） 文档分类

文档多了全在Home下，好几屏，挺烦人，用kindle自带的功能即可轻松分类，[Home] / [Menu] / [Create new collection]，进去创建好的Collection后按 [Menu] / [Add/Remove Items]可以批量操作分类，Collection名我全用的英文或汉语拼音，中文输入法现在觉得没装的必要性。另：输入数字可以按住Alt键盲按QW……OP，分别对应12……90，比用Sym键方便些。

仅仅分好类会发现文档不仅出现在collection里，在Home里还是照常出现，还需要操作一下：[Home] / 5项键上箭头到顶 [Showing All xxx Items] / 右箭头到 [By Collections] 即可。

（六） 破解wifi传文件

你要不想整天带个数据线或者不想email文章绕到 @free.kindle.com再回来，这一步也是必须的，破解了蛋疼的话还可以ssh登录kindle敲敲linux命令。方法为：

首先把上面下载的kindle-usbnetwork-0.34.N.zip解压后把 update_usbnetwork_0.34.N_k3g_install.bin拷贝到kindle根目录下，在kindle上按[Home] / [Menu] / [Settings] / [Menu] / [Update your kindle]即可，重启较慢，耐心等待。重启后还要设置一下，最后一次用USB线连Kindle，把 /usbnet/DISABLED_auto 文件改名为auto，修改/usbnet/etc/config文件中两个false为true：

K3_WIFI="true"

K3_WIFI_SSHD_ONLY="true"

之后到http://winscp.net 下载个winscp装上，我刚好前两天下了个来同步Amazon EC2的Linux虚拟机，就把这步省了，话说Amazon真是个严肃靠谱的美企，老出一些改变人类生活的好东西。安装后用winscp带的puttygen.exe来Generate一个公钥和一个私钥，把公钥文本粘贴保存到kindle的/usbnet/etc/ authorized_keys文件中（注意用unix文件格式）,私钥随便起个名字如xxx4winscp.ppk保存在PC机上备用，kindle上按[Home] / [Menu] / [Settings] / [Menu] / [Restart]重启。

然后在kindle上打开wifi，用 [HOME] / [MENU] / [Settings]，输入 Alt+u, Alt+q, Alt+q,翻到第二页就可看到IP地址了，在不同wifi热点例如公司和家里IP地址略有不同，可以在winscp里面建多个登录会话备用，要正常看中文文件名需要用SFTP协议并开启文件名UTF-8编码。以后就可以少用或者不用数据线了。另：从winscp中按Ctrl-P可以直接登录到kindle的linux console下。还有一个重要的设置：原来通过usb数据线copy文件，拔下来之后kindle自动在首页刷新文档列表，但用winscp传文件后kindle不知道你传了，需要人工执行命令告诉它，在linux console下敲：

dbus-send --system /default com.lab126.powerd.resuming int32:1

或者把以上命令写在/usbnet/refresh_kindle文件内，通过winscp在该文件上点 [右键] / [自定义命令] / [执行] 来手工刷新Home的列表。

好了，先折腾了这些，以后有新的体会再补充。
